# React + TypeScript + Vite 项目指南
# 🚀 快速开始

## 2. 安装项目依赖
``` 
    // 使用 pnpm 安装（推荐）
    pnpm install
    // 如果未安装 pnpm，先运行：
    npm install -g pnpm@latest
```

## 3. 启动开发服务器
```
    // 运行开发模式
    pnpm run dev

    # ⚠️ 如果 PowerShell 报错 "拒绝执行脚本"：

    # 解决方案-1
    # 在 setting.json 中添加："terminal.integrated.defaultProfile.windows": "Command Prompt"

    # 解决方案-2：
    # 1. 以管理员身份运行 PowerShell
    # 2. 执行：：Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
    # 3. 选择 [Y] 确认更改
```
